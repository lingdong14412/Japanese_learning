# 日语动词分类与时态变形规律

## 一、动词分类

日语动词按活用方式分为 **三大类**：

### 1. 五段动词（一类动词 / u动词）

- **特征**：词尾在「う段」上，且最后一个假名为「う、く、す、つ、ぬ、む、る、ぐ、ぶ」
- **判断方法**：不以「る」结尾，或以「る」结尾但「る」前的假名不在「い段/え段」
- **例子**：

| 原形 | 含义 | 词尾 |
|------|------|------|
| 書く（かく） | 写 | く |
| 読む（よむ） | 读 | む |
| 飲む（のむ） | 喝 | む |
| 話す（はなす） | 说 | す |
| 待つ（まつ） | 等 | つ |
| 死ぬ（しぬ） | 死 | ぬ |
| 遊ぶ（あそぶ） | 玩 | ぶ |
| 帰る（かえる） | 回去 | る（る前为え段 → 五段） |

### 2. 一段动词（二类动词 / ru动词）

- **特征**：以「る」结尾，且「る」前的假名在「い段」或「え段」
- **细分**：
  - **上一段**：「る」前为「い段」假名（見る、起きる、食べる→不对，食べる是下一段）
  - **下一段**：「る」前为「え段」假名（食べる、寝る、教える）

| 类型 | 例子 | 含义 |
|------|------|------|
| 上一段 | 見る（みる） | 看 |
| 上一段 | 起きる（おきる） | 起床 |
| 上一段 | いる | 在/有（有生命） |
| 下一段 | 食べる（たべる） | 吃 |
| 下一段 | 寝る（ねる） | 睡觉 |
| 下一段 | 教える（おしえる） | 教 |

> **注意**：存在少数特殊五段动词，形式上像一段但实际是五段，需单独记忆：
> 帰る（かえる）、切る（きる）、知る（しる）、走る（はしる）、入る（はいる）等

### 3. 不规则动词（三类动词）

只有 **两个**：

| 动词 | 含义 |
|------|------|
| する | 做 |
| 来る（くる） | 来 |

## 二、时态变形总览

日语动词有 **六种基本活用形**，是所有时态/语法的基础：

| 活用形 | 作用 |
|--------|------|
| 未然形 | 否定、被动、使役 |
| 連用形 | ます形、て形、过去形的基础 |
| 終止形 | 结句（原形/辞书形） |
| 連体形 | 修饰名词 |
| 仮定形 | 假定（～ば） |
| 命令形 | 命令 |

---

## 三、各变形规则详解

### 1. ます形（連用形 + ます）

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → い段 + ます | 書く → 書**き**ます |
| 一段 | 去「る」 + ます | 食べる → 食べ**ます** |
| する | します | する → **します** |
| 来る | 来ます（きます） | 来る → **来ます** |

### 2. 否定形（未然形 + ない）

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → あ段 + ない | 書く → 書**かない** |
| 一段 | 去「る」 + ない | 食べる → 食べ**ない** |
| する | しない | する → **しない** |
| 来る | 来ない（こない） | 来る → **来ない** |

> **特殊**：「う」结尾的五段动词 → **わ**ない（買う → 買**わ**ない，不是買あない）

### 3. て形（連用形 + て）

**五段动词**（音便规则）：

| 词尾 | 音便 | 例子 |
|------|------|------|
| く | いて | 書く → 書**いて** |
| ぐ | いで | 泳ぐ → 泳**いで** |
| す | して | 話す → 話**して** |
| つ/る/う | って | 待つ → 待**って** / 帰る → 帰**って** / 買う → 買**って** |
| む/ぶ/ぬ | んで | 読む → 読**んで** / 遊ぶ → 遊**んで** / 死ぬ → 死**んで** |

> **特殊**：行く → 行**って**（不是行いて）

**一段/する/来る**：

| 类型 | 规则 | 例子 |
|------|------|------|
| 一段 | 去「る」 + て | 食べる → 食べ**て** |
| する | して | する → **して** |
| 来る | 来て（きて） | 来る → **来て** |

### 4. 过去形（た形）

**规则与て形完全相同**，只需把「て→た」「で→た」：

| て形 | → | た形 |
|------|---|------|
| 書いて | → | 書いた |
| 泳いで | → | 泳いだ |
| 話して | → | 話した |
| 待って | → | 待った |
| 読んで | → | 読んだ |
| 食べて | → | 食べた |
| して | → | した |
| 来て（きて） | → | 来た（きた） |

### 5. 可能形

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → え段 + る | 書く → 書**ける** |
| 一段 | 去「る」 + られる | 食べる → 食べ**られる** |
| する | できる | する → **できる** |
| 来る | 来られる（こられる） | 来る → **来られる** |

> **口语简略**：一段动词的「られる」常省略为「れる」（食べれる），但书面语中被视为不规范。

### 6. 被动形（受身形）

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → あ段 + れる | 書く → 書**かれる** |
| 一段 | 去「る」 + られる | 食べる → 食べ**られる** |
| する | される | する → **される** |
| 来る | 来られる（こられる） | 来る → **来られる** |

### 7. 使役形

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → あ段 + せる | 書く → 書**かせる** |
| 一段 | 去「る」 + させる | 食べる → 食べ**させる** |
| する | させる | する → **させる** |
| 来る | 来させる（こさせる） | 来る → **来させる** |

### 8. 意志形（～う/よう）

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → お段 + う | 書く → 書**こう** |
| 一段 | 去「る」 + よう | 食べる → 食べ**よう** |
| する | しよう | する → **しよう** |
| 来る | 来よう（こよう） | 来る → **来よう** |

### 9. 假定形（～ば）

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → え段 + ば | 書く → 書**けば** |
| 一段 | 去「る」 + れば | 食べる → 食べ**れば** |
| する | すれば | する → **すれば** |
| 来る | 来れば（くれば） | 来る → **来れば** |

### 10. 命令形

| 类型 | 规则 | 例子 |
|------|------|------|
| 五段 | 词尾 → え段 | 書く → 書**け** |
| 一段 | 去「る」 + ろ（/よ） | 食べる → 食べ**ろ** |
| する | しろ / せよ | する → **しろ** |
| 来る | 来い（こい） | 来る → **来い** |

## 四、速查表（以「書く」「食べる」「する」「来る」为例）

| 变形 | 書く（五段） | 食べる（一段） | する | 来る |
|------|-------------|---------------|------|------|
| ます形 | 書きます | 食べます | します | 来ます |
| 否定 | 書かない | 食べない | しない | 来ない |
| て形 | 書いて | 食べて | して | 来て |
| た形 | 書いた | 食べた | した | 来た |
| 可能 | 書ける | 食べられる | できる | 来られる |
| 被动 | 書かれる | 食べられる | される | 来られる |
| 使役 | 書かせる | 食べさせる | させる | 来させる |
| 意志 | 書こう | 食べよう | しよう | 来よう |
| 假定 | 書けば | 食べれば | すれば | 来れば |
| 命令 | 書け | 食べろ | しろ | 来い |

## 五、快速判断口诀

```
不以る结尾 → 五段
以る结尾 + い/え段前 → 一段（排除特殊五段）
する/来る → 不规则
```

> **核心规律**：五段动词词尾在「あ/い/う/え/お」五段之间变化（故名"五段"），
> 一段动词只需去掉「る」再加词尾（故名"一段"）。
