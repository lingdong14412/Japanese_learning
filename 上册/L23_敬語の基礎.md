# 第23课 敬语基础

## 核心语法

### 1. 尊敬语（对对方/他人的动作表示尊敬）

| 普通形 | 尊敬语 |
|--------|--------|
| いる | いらっしゃいます |
| 行く | いらっしゃいます |
| 来る | いらっしゃいます |
| する | なさいます |
| 食べる | 召し上がります（めしあがります） |
| 飲む | 召し上がります |
| 言う | おっしゃいます |
| 見る | ご覧になります（ごらんになります） |
| 知る | ご存知です（ごぞんじです） |

### 2. お＋动词连用形＋になります（尊敬语句型）

> 先生はもうお帰りになりました。（老师已经回去了。）

### 3. 谦让语（降低自己以抬高对方）

| 普通形 | 谦让语 |
|--------|--------|
| いる | おります |
| 行く | 参ります（まいります） |
| 来る | 参ります |
| する | いたします |
| 食べる | いただきます |
| 言う | 申します（もうします） |
| 見る | 拝見します（はいけんします） |

### 4. お＋动词连用形＋します（谦让语句型）

> お荷物をお持ちします。（我来帮您拿行李。）

### 5. 丁寧语（礼貌语）
`です`/`ます`体即为最基本的礼貌语。

---

## 补充要点
- 尊敬语用于抬高对方的动作；谦让语用于降低自己的动作。
- 不规则变化的尊敬语/谦让语需要单独记忆。
- `お～になる`是尊敬语，`お～する`是谦让语。
