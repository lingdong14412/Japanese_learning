# 第12课 日期与星期

## 核心语法

### 1. 日期表达

| 表达 | 读法 |
|------|------|
| 一日 | ついたち |
| 二日 | ふつか |
| 三日 | みっか |
| 四日 | よっか |
| 五日 | いつか |
| 六日 | むいか |
| 七日 | なのか |
| 八日 | ようか |
| 九日 | ここのか |
| 十日 | とおか |
| 二十日 | はつか |

### 2. ～月（がつ）
一月～十二月。

### 3. ～曜日
月曜日～日曜日（周一～周日）。

### 4. ～に（时间点）

> 四月に日本へ行きます。（四月去日本。）
> 月曜日に会います。（周一见面。）

**注意**：`今日`、`明日`、`昨日`、`来週`、`先週`等时间词后不加`に`。

### 5. いつ（什么时候）

> 誕生日はいつですか。（生日是什么时候？）
> 誕生日は三月十日です。（生日是三月十号。）

### 6. 頻度副词

| 副词 | 含义 |
|------|------|
| いつも | 总是 |
| よく | 经常 |
| 時々 | 有时 |
| たまに | 偶尔 |
| あまり | 不太 |
| ぜんぜん | 完全不 |

---

## 补充要点
- 日语日期中1～10号有特殊读法，需要逐一记忆。
- ～日（にち）用于读日期中的天数。
