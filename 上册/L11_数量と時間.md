# 第11课 数量与时间表达

## 核心语法

### 1. 数量词的用法
数量词放在动词前面。

> りんごを三個食べました。（吃了三个苹果。）

### 2. 常用量词

| 量词 | 对象 | 例 |
|------|------|---|
| ～個 | 小物品 | 一個、二個 |
| ～人 | 人 | 一人、二人 |
| ～枚 | 薄平物 | 一枚、二枚 |
| ～本 | 细长物 | 一本、二本 |
| ～杯 | 杯装饮品 | 一杯、二杯 |
| ～台 | 机器 | 一台、二台 |
| ～匹 | 小动物 | 一匹、二匹 |
| ～冊 | 书本 | 一冊、二冊 |

### 3. 时间表达

> 毎日八時間寝ます。（每天睡八小时。）

### 4. ～に（时间点）
表示动作发生的时间点。

> 七時に起きます。（七点起床。）
> 月曜日に会社へ行きます。（周一去公司。）

### 5. ～から～まで（从……到……）

> 九時から五時まで働きます。（从九点工作到五点。）

---

## 补充要点
- 日语量词随数字不同会发生音变（如：いっぱいい、にほん、さんぼん）。
- `毎`（まい）不加`に`：毎日、毎週、毎月、毎年。
