# 第36课 语法知识点

## 1. 形容词的て形

表示形容词的并列。

- **い形容词**：去掉い＋くて
  - 高い → 高くて（又高又～）
  - 静か → 静かで（又安静又～）

- **な形容词**：词干＋で
  - 静か → 静かで（又安静又～）
  - きれい → きれいで（又漂亮又～）

## 2. 形容词并列的用法

- この部屋は広く**て**明るいです。（这个房间又宽敞又明亮。）
- その公園は静か**で**美しいです。（那个公园又安静又美丽。）

## 3. 形容词的名词化

- い形容词：去掉い＋さ
  - 高い → 高さ（高度）
  - 暑い → 暑さ（热度）

- な形容词：词干＋さ
  - 便利 → 便利さ（便利程度）
