# 第35课 语法知识点

## 1. ～を～ます（他动词）

他动词表示人为的动作，需要宾语。

- 窓を**開けます**。（打开窗户。）
- 電気を**消します**。（关灯。）

## 2. ～が～ています（状态）

表示动作结果的状态。

- 窓が**開いています**。（窗户开着。）
- 電気が**ついています**。（灯亮着。）

## 3. 自动词和他动词的区别

| 自动词（状态） | 他动词（动作） |
|---------------|---------------|
| 開きます（开） | 開けます（打开） |
| 閉まります（关） | 閉めます（关上） |
| 始まります（开始） | 始め点（开始做） |
| 終わり点（结束） | 終えます（做完） |

- ドアが**開いています**。（门开着。）——状态
- ドアを**開けます**。（把门打开。）——动作
